## Requirements

- bash
- gnu grep, either as `grep` or `ggrep`
- neovim 0.7.0+
- Hasklug Nerd Font
  - `brew install font-hasklug-nerd-font`
- [nonicons font](https://github.com/yamatsum/nonicons/blob/master/dist/nonicons.ttf)
- <https://github.com/savq/paq-nvim#installation>

## Installation

```sh
./install.bash
```
